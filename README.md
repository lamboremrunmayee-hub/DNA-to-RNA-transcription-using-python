# DNA-to-RNA-transcription-using-python
A Python program to convert DNA sequence to RNA sequence.

#Code
DNA=input(str("Enter the DNA sequence - "))
RNA=DNA.replace('T','U')
print("The RNA sequence of the given DNA sequence is :",RNA)

#Output
Enter the DNA sequence - ATCGTCG
The RNA sequence of the given DNA sequence is - AUCGUCG

## How to Run
1.Install Python
2. Save the code in a file
3. Run the program:
   python dna_to_rna.py
